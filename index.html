This program is a core software practice for object-oriented GIS software development lecture.

This program is a core software practice for object-oriented GIS software development lecture. It can encode and decode text. It can open and write .shp, .dbf, and .geojson. It also can use .sld to render layer style. In a word, it is a easy-to-use software.

Feature

  1. Easy to use and similar to qgis operations

  2. Cross Platform

  3. Supported vectorfeature formats include:

    • ESRI Shapefiles

    • GeoJson

  4. Supported style formats include:

    • sld
